Clear Cell Game

A Class Project

This is a Java project that was written in the Eclipse 17.0.2 IDE.

It was nice to have some overlap between my academic and personal interests. The logic needed to change the color and position of each block is actually very similar to a flag drawing application I developed during the previous semester, but this time I can interact with the blocks. This row/col manipulation is done through interactions between 2D arrays, an Interface, and Enumerated Type. The game is like reverse-Tetris, and is actually pretty addicting (my high score is 927).

If you are interest in viewing the code of this project, as well as others, please browse my GitHub.
